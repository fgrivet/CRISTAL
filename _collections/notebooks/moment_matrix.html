Moment Matrix Example#

Imports#

import numpy as np

import cristal as cst

Parameters#

d = 8  # Data dimension
n = 5  # Degree of the polynomial basis
N = 1000  # Number of samples

Data#

np.random.seed(42)
data = np.random.random((N, d))

Method 1: Using DyCF#

Create a DyCF instance#

dycf = cst.DyCF(n, inverter_opt=cst.IMPLEMENTED_INVERTERS.PSEUDO)

Fit the DyCF model to the data to compute the moments matrix#

dycf.fit(data)

<cristal.detectors.datasteams.DyCF at 0x7fda18fb1be0>

Access the moments matrix and its inverse#

moments_matrix = dycf.moments_matrix.moments_matrix
inversed_moments_matrix = dycf.moments_matrix.inverse_moments_matrix

Method 2: By computing the design matrix X#

Generate the multidegree combinations#

multidegree_combinations = cst.MultivariatePolynomialBasis.generate_multidegree_combinations(n, d)

Compute the design matrix X#

X = cst.MultivariatePolynomialBasis.make_design_matrix(data, multidegree_combinations, cst.IMPLEMENTED_POLYNOMIALS.MONOMIALS.value)

Construct the moments matrix M and its inverse M_inv#

M = np.dot(X.T, X) / N
M_inv = cst.IMPLEMENTED_INVERTERS.PSEUDO.value.invert(M)

Equality check#

np.allclose(moments_matrix, M, atol=1e-16), np.allclose(inversed_moments_matrix, M_inv, atol=1e-16)

(True, True)
