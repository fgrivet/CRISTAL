Installing CRISTAL
===================

To install CRISTAL, activate your virtual environment (optional) and use the following command:

.. code-block:: bash

   git clone https://github.com/fgrivet/CRISTAL.git
   cd CRISTAL
   pip install -e .


Or 

.. code-block:: bash

   pip install git+https://github.com/fgrivet/CRISTAL.git
