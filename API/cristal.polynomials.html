cristal.polynomials#

CRISTAL Polynomials Subpackage Contains various polynomial classes, including Monomials, Chebyshev and Legendre polynomials.

class cristal.polynomials.IMPLEMENTED_POLYNOMIALS(*values)#

Bases: Enum

The implemented polynomial classes.

CHEBYSHEV_T1 = <class 'cristal.polynomials.chebyshev.ChebyshevT1Family'>#
CHEBYSHEV_T2 = <class 'cristal.polynomials.chebyshev.ChebyshevT2Family'>#

Modules#

base

Base classes for polynomial representations (BasePolynomialFamily) and for multivariate polynomial basis generation (MultivariatePolynomialBasis).

chebyshev

Chebyshev polynomial families of the first and second kind.

legendre

Legendre polynomial family.

monomials

Monomials polynomial family.
